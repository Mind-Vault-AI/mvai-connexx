# Lessons Learned — mindvault-ai.com

## Sprint 1 (2026-03-12)

### Wat werkte goed
- Bestaande landing.html als stijlreferentie → veel tijd bespaard
- Werkwijze framework opgesteld voor alle toekomstige projecten
- Honeypot + GDPR ingebouwd vanaf het begin (niet achteraf)

### Verbeterpunten
- Hostinger API was niet bereikbaar → altijd fallback plan voor DNS/infra wijzigingen
- Preview-first aanpak = goed: review vóór deployment voorkomt wasted werk

### Beslissingen
- Engels als standaard taal (global rollout)
- Render + Hostinger only (geen extra abonnementen)
- Newsletter data in Connexx DB → CSV export naar BOX Drive
